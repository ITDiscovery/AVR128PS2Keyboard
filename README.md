# AVR128PS2Keyboard
Re-writing the PS2/USB Keyboard reader on the AVR

https://www.youtube.com/watch?v=7aXbh9VUB3U

Looks like the PS2KeyAdvanced has been updated.
https://github.com/techpaul/PS2KeyAdvanced
